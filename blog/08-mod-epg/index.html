提供使用者快速、無痛的方式找到所需資訊。

EPG使用情境

在開始專案之前，我們深入調查了和訪談了30位左右的現有MOD頻道的用戶，發現在節目指南 (EPG) 的使用情境中，存在以下5種主要情況：

1. 隨性搜索

當使用者不確定要觀看什麼節目時，他們會拿起遙控器，點擊搜索按鈕，開始瀏覽節目清單。透過關鍵字搜索，使用者可以尋找引起興趣的節目，這種隨性的搜索能帶來驚喜，發現了之前未注意到的優秀節目。

2. 頻道名稱搜索

使用者有時會記得特定頻道的名稱，但忘記了頻道號碼。在這種情況下，使用者可以使用遙控器上的搜索功能，輸入頻道名稱或部分關鍵詞，系統會列出相關的頻道選項，然後使用者只需點選正確的頻道，輕鬆找到想要的節目。

3. 播出時間搜索

當使用者想觀看特定節目但忘記了播出時間和頻道時，他們會使用電視節目指南或網絡搜索來查找該節目的播出時間和頻道，確保不錯過喜愛的節目。

4. 頻道異動搜索

當頻道或播出時間發生異動，使用者會嘗試搜索節目名稱，以查找它是否轉移到其他頻道或播出時間，確保不錯過任何一集。

5. 系列節目追蹤

對於使用者喜愛的系列節目，如戲劇或綜藝節目，他們通常會設定提醒功能，以便記住未來的播出時間，確保不錯過任何一集。

改善痛點與機會點

在了解各種EPG使用情境後，我們發現了設計上的機會和挑戰。為了提高使用體驗，我們針對每種情境的痛點提供簡潔且直觀的解決方案，並聚焦於簡化操作、提供清晰的內容和指引，確保使用者能輕鬆找到所需的節目。

痛點

  1. 操作不易：目前介面與遙控器操作不直覺，使用方向鍵選取不方便，容易誤導使用者操作。
  2. 階層不明確：操作選單的層級未區分，選單內容有許多和節目無關的操作。
  3. 資訊顯示不足：節目名稱與節目內容資訊不足，無法清晰判斷節目內容。

機會點

  1. 改善查找方式：引入漏斗式篩選來過濾查找節目，例如類別 > 頻道 > 節目，以加速縮小搜尋範圍，簡化操作。
  2. 加強內容提供：主動告知該節目在MOD上的串流影視收看渠道，並實現交叉查找功能，使使用者更容易找到所需節目，並引導付費訂閱。
  3. 資訊一目瞭然：加強文字閱讀性，改善排版、動線和階層，提供更完整的節目資訊。

改善方案

為實現上述機會點，我們提出以下改善方案：

  1. 簡化操作： 精簡遙控器功能鍵，使操作更直觀。
  2. 強調台號： 突顯頻道號碼，讓使用者更容易找到他們想要的頻道。
  3. 提供完整節目資訊： 增加節目名稱、內容資訊等細節，讓使用者能夠清晰了解節目內容。
  4. 區分層級： 明確標示操作選單的層級，並減少無關操作的干擾。
  5. 所在提示： 提供清晰的所在位置提示，以防使用者迷失在介面中。
  6. 功能整合與加強： 整合預約和預錄功能，實現交叉銷售。

設計重點與交付

🔍 漏斗式篩選

在重新設計MOD頻道節目表時，我們的主要設計重點是提供使用者一個快速、直觀且無痛的方式找到他們所需的資訊。

  1. 漏斗式階層查找： 加速縮小搜尋範圍，類別>頻道>節目 立即幫助找節目
  2. 有效溝通節目內容，資訊呈現更清晰： 加入海報與詳細節目內容，方便判斷
  3. 交叉銷售、其他管道不錯過： 一次列出該節目可收視管道，滿足用戶立即觀看慾望
  4. 頻道VOD化： 該節目不同時段錄影、回看、套餐資訊一目了然

改善成果

  1. 使用者將能更輕鬆地找到他們感興趣的節目，提高了整體用戶滿意度。
  2. 減少使用者迷失。
  3. 增加了交叉隨選影視的營收管道。

總結

重新設計MOD頻道節目表是一個針對不同EPG使用情境的挑戰性任務。通過專注於簡化操作、提供清晰的資訊和改善界面設計，我們的目標是為使用者提供更好的觀影體驗。我們期待這些改善將提高用戶滿意度，使MOD頻道成為優質的娛樂選擇，滿足各種觀看需求。
